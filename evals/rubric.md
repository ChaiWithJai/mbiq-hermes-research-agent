# Job-level evaluation

Score each case from 0 to 2.

- 0: missing, fabricated, or unusable.
- 1: partly complete but needs a researcher to redo material work.
- 2: meets every listed pass condition and is ready for editorial review.

The system passes only if every case scores 2. This is deliberately stricter than checking whether the model can emit a tool call. The target is a meaningful economic job: an editor receives a trustworthy packet and can move directly into reporting or story development.
